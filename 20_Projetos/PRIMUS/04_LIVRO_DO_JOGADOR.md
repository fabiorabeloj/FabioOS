# BLOCO 4 — Livro do Jogador (O que o Jogador Usa)

**Objetivo:** Permitir criação de personagem e jogo rápido.

---

## 📖 Conteúdo Mínimo (Preenchido depois)

### Seção 1: Raças
**Template:** RACE (de BLOCO 3)

Adicionar aqui:
- [ ] Humano
- [ ] Elfo
- [ ] Anão
- [ ] Halfling
- [ ] (Dragonborn, Tiefling, Gnome, Half-Elf, Half-Orc, conforme necessário)

**Cada raça deve ter:**
- Traços
- Nomes típicos
- Bônus de atributos
- Tamanho, velocidade
- Linguagens

---

### Seção 2: Classes
**Template:** CLASS (de BLOCO 3)

Adicionar aqui:
- [ ] Mago
- [ ] Guerreiro
- [ ] Clérigo
- [ ] Ladrão
- [ ] (Bardo, Druida, Monge, Paladino, conforme necessário)

**Cada classe deve ter:**
- Hit die
- Proficiências
- Features por nível
- Spellcasting (se houver)

---

### Seção 3: Antecedentes
**Template:** BACKGROUND (variação de RACE)

Adicionar aqui:
- [ ] Soldado
- [ ] Mercador
- [ ] Criminoso
- [ ] Acolito
- [ ] (Charlatão, Guildmember, Folk Hero, conforme necessário)

**Cada antecedente deve ter:**
- Habilidades garantidas
- Traço pessoal
- Ideal
- Vínculo
- Flaw

---

### Seção 4: Equipamento
**Template:** ITEM + MAGIC_ITEM (de BLOCO 3)

Lista de:
- Armas comuns (adaga, espada, arco, etc)
- Armaduras (couro, cota de malha, placa, etc)
- Itens de adventureiro (corda, mochila, lanterna, etc)
- Itens mágicos (conforme disponível)

---

### Seção 5: Talentos
**Template:** FEAT (de BLOCO 3)

Adicionar:
- [ ] Ataque Extra
- [ ] Esquiva
- [ ] Magia de Ação
- [ ] Great Weapon Master
- [ ] (Conforme necessário)

**Cada talento deve ter:**
- Pré-requisito (se houver)
- Efeito mecânico
- Quando pode usar

---

### Seção 6: Magias
**Template:** SPELL (de BLOCO 3)

Organize por:
- Nível (0, 1, 2, 3, etc)
- Escola (Evocação, Abjuração, etc)

**Cada magia deve ter:**
- Nível de lançamento
- Alcance
- Componentes
- Duração
- Efeito

---

### Seção 7: Regras Mínimas de Criação
```
1. Escolha raça [[race]]
2. Escolha classe [[class]]
3. Escolha antecedente [[background]]
4. Role atributos: 4d6 (descartar menor)
5. Aplique bônus de raça
6. Escolha equipamento
7. Escolha talentos (conforme classe permite)
8. Calcule CA, HP, bônus de ataque
9. Pronto para jogar!
```

---

## 🔗 Integração com Blocos Anteriores

- **Bloco 1** define: Circuito E→I→P
- **Bloco 2** define: Tipos (race, class, background, etc existem)
- **Bloco 3** define: Templates (como race, class se estruturam)
- **Bloco 4** (ESTE): Preenche templates com conteúdo do jogador
- **Bloco 5** usa: Este livro para criar instâncias
- **Bloco 6** testa: Se criação funciona
