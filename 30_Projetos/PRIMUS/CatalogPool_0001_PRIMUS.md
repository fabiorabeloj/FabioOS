# CatalogPool 0001

## Funcao

Agrupar entradas candidatas para futura Engrenagem 6 sem executar missao.

## Pool Atual

| Grupo | Entries | Uso |
|---|---|---|
| Jogador | CE-DND-0001 a CE-DND-0006 | personagem, recursos e efeitos |
| Desafio | CE-DND-0007 a CE-DND-0010 | encontro, criatura, condicao, recompensa |
| Mundo | CE-WWN-0001, CE-WWN-0002, CE-WWN-0005, CE-WWN-0006 | lugar, hub, site, dungeon |
| Agentes coletivos | CE-WWN-0003, CE-WWN-0009 | faccao, patrono |
| Gatilhos | CE-WWN-0004, CE-WWN-0007, CE-WWN-0008, CE-WWN-0010 | tags, eventos, problemas, hooks |

## Bloqueio

Nenhuma entrada deste pool esta `pool-ready` ate passar por V(E). O pool serve para organizar a validacao, nao para gerar missao ainda.
