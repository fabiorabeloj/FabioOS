# Respostas-Modelo — Pietra

## Como usar

Cada resposta-modelo é um rascunho editável. O professor deve:
1. Ler o rascunho sugerido pelo Pietra
2. Personalizar com nome do aluno, data ou informação específica
3. Aprovar e enviar pelo canal adequado

**Nunca enviar diretamente sem revisão.**

Substituições comuns: `[NOME_ALUNO]`, `[TURMA]`, `[DATA]`, `[DISCIPLINA]`, `[HORÁRIO]`, `[CONTATO]`.
