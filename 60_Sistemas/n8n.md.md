
# n8n

## Função

Motor de automação do FabioOS.

## Objetivo

Executar processos repetitivos e integrar ferramentas sem exigir ação manual constante.

## Responsabilidades

- Ler e-mails.
    
- Ler agenda.
    
- Processar formulários.
    
- Atualizar planilhas.
    
- Acionar modelos de IA.
    
- Criar documentos.
    
- Enviar notificações.
    
- Executar workflows.
    

## O que ele NÃO é

- Não é memória.
    
- Não é inteligência.
    
- Não é repositório de conhecimento.
    
- Não é sistema de decisão estratégica.
    

## O que ele É

Um executor de processos.

## Relação com os outros sistemas

### Obsidian

Função:  
Memória.

Fluxo:  
n8n pode registrar informações no Obsidian.

### GitHub

Função:  
Versionamento.

Fluxo:  
n8n pode armazenar workflows e documentação.

### Claude / GPT

Função:  
Inteligência.

Fluxo:  
n8n envia contexto e recebe análise.

### OpenClaw

Função:  
Coordenação.

Fluxo:  
OpenClaw poderá acionar workflows do n8n.

## Workflows Planejados

### Escola

- Identificar tarefas.
    
- Registrar pendências.
    
- Gerar notificações.
    

### FabioOS

- Registrar ideias.
    
- Organizar informações.
    
- Atualizar dashboards.
    

### Agenda

- Monitorar eventos.
    
- Detectar conflitos.
    
- Criar lembretes.
    

### Conhecimento

- Processar notas.
    
- Organizar informações.
    
- Atualizar registros.
