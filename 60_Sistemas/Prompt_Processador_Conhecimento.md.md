# Prompt — Processador de Conhecimento

Transforme o conteúdo abaixo em uma nota Markdown pronta para o Obsidian.

## Regras
1. Crie um título claro.
2. Faça um resumo objetivo.
3. Identifique a ideia central.
4. Organize o desenvolvimento.
5. Extraia aplicações práticas.
6. Extraia tarefas em checklist.
7. Extraia decisões importantes.
8. Adicione links internos no formato [[Nome da Nota]].
9. Adicione tags.
10. Sugira a pasta ideal dentro do FabioOS.
