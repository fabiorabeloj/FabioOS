# Sources — FabioOS

## Função

Fontes brutas que alimentam a wiki do FabioOS. Aqui ficam resumos controlados de repositórios, artigos, PDFs transcritos, prints, notas originais e materiais de referência.

## Subdiretórios

```
sources/
  repositorios/    ← nota-fonte de cada repo clonado em claude-extensions/
  artigos/         ← resumos de artigos e posts relevantes
  pdfs/            ← transcrições e resumos de PDFs
  aulas/           ← materiais de aulas e cursos
```
