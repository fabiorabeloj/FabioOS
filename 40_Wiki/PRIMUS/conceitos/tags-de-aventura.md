# Tags de Aventura

## O conceito
Cada lugar/situação recebe uma ou duas **tags** (ex.: "colônia perdida", "culto
oculto", "guerra fria"). Cada tag traz um **kit pronto**: aliados típicos, inimigos,
complicações, coisas (itens/objetivos) e lugares. A tag é um **motor de ideias** —
não descreve tudo, mas dá ganchos suficientes para improvisar uma aventura.

## Por que é forte
- **Gera** conteúdo em vez de listar: pouca entrada, muita saída.
- **Abstrata e reutilizável:** vale para qualquer cenário.
- **Ligável:** uma tag alimenta tensões, facções e conflitos.

## Relações no grafo
- Fornece complicações à [[faccoes-e-turnos-de-faccao|dinâmica de facções]].
- Vira insumo de [[Spec_Tension_Engine_PRIMUS|tensões]] e de conflitos da
  [[Spec_Cantina_Conflict_Engine_PRIMUS|Cantina]].
- Cada tag aplicada é uma variacao do mundo -> [[Spec_DeltaP_PRIMUS|DeltaP]].

## Onde o DADO vive
Listas específicas de tags de um livro → índice/PRIMUS Index. O grafo fica só com o
**conceito de tag** e as tags autorais que o Fabio criar.

## Uso no PRIMUS
Dar 1-2 tags a cada região/local do [[WorldState_0001_PRIMUS]]; usar os kits para
gerar ganchos rápidos ao montar uma cena.
