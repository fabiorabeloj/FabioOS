# Enciclopedia PRIMUS

## Funcao

Camada E do PRIMUS: conhecimento canonico, validado e reutilizavel. Aqui entram raças, classes, itens, faccoes, lugares, criaturas, regras e conceitos do mundo.

## Regras

- Nao receber texto integral de fontes restritas.
- Cada entrada deve ter `primus_id`, tipo, status, fonte e licenca.
- Cada entrada deve linkar [[40_Wiki/_MOCs/MOC_PRIMUS]].
- Entradas derivadas de D&D ficam como resumo proprio e referencia, nunca como copia.
