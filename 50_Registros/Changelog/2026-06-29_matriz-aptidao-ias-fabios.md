# Changelog - Matriz de Aptidao das IAs FabioOS

## Resumo

Criada a matriz operacional que define vocacao, tarefas recomendadas, tarefas proibidas, papel, integracao, risco, teste e criterio de permanencia das IAs, modelos e ferramentas citadas no FabioOS.

## Entregas

- `90_Arquivo/Legado_Pre_LLM_Wiki_2026-06-29/40_Repertorio/IA/Modelos_e_IAs/Matriz_de_Aptidao_das_IAs_FabioOS.md`
- Atualizacoes em STATUS, NEXT_ACTIONS, mapa, roteamento, specs, skills, agentes e prompt de orquestracao.

## Decisao

O FabioOS nao tratara IAs como equivalentes. Cada IA/modelo/ferramenta deve existir por competencia especifica, por reducao real de custo/risco ou por redundancia estrategica testada.

## Limites

- Nenhuma API externa foi chamada.
- Nenhum token foi criado, solicitado ou salvo.
- Nenhuma automacao externa foi executada.
- Nenhum runtime OpenClaw/n8n/MCP foi alterado.

## Proxima acao

Rodar testes pequenos de aptidao antes de promover qualquer IA de `observacao` para `piloto` ou `ativo`.
