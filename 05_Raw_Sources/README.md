# 05_Raw_Sources

Camada de fontes brutas do FabioOS.

Aqui entram evidencias originais: PDFs, prints, cursos, transcricoes, conversas exportadas, documentos e materiais de radar tecnologico.

Regra:

```text
Fonte bruta e evidencia. Nao editar destrutivamente.
```

Compatibilidade:

`sources/` continua existindo como pasta operacional legada enquanto RAG, MCP, scripts e links sao migrados por lote.
