## Decisao

Os tres PDFs D&D Core enviados pelo Fabio entram no PRIMUS como **fontes restritas**, nao como texto a ser copiado para Obsidian, Git ou RAG.

```text
PDF Restricted -> metadados -> CatalogEntry -> curadoria -> uso privado controlado
```

## Uso Permitido

- Validar CatalogEntries privadas.
- Guardar metadados e referencias de pagina.
- Criar resumos funcionais de alto nivel.
- Criar ponte para o PRIMUS Index local.
- Usar pequenos snippets apenas como rastreio privado, se necessario.

## Uso Proibido

- Copiar capitulos ou secoes integrais.
- Exportar texto para Git.
- Publicar conteudo.
- Reindexar texto bruto no RAG.
- Transformar regra comercial em regra base do PRIMUS sem decisao de precedencia.

## Resumo Funcional Seguro

### SRC-DND-PHB-2014

Fonte de regras de personagem. Serve para validar entradas sobre classe, especie/raca, antecedente, magia, equipamento, condicoes e escolhas de personagem.

Fonte de regras de mestre. Serve para validar entradas sobre encontro, desafio, tesouro, itens magicos, estrutura de aventura e consequencias de jogo.

### SRC-DND-MM-2014

Fonte de criaturas. Serve para validar entradas sobre bestiario, antagonistas, ameacas, papeis de combate e referencias de criatura.
