# Eventos do Drop PDF

Cada evento representa um PDF detectado pelo watcher local.

O evento nao significa que o documento ja foi ingerido no RAG. Ele significa:

```text
PDF detectado -> pronto para documentalista/Claude processar quando o Stirling estiver operacional
```

Arquivos `.json` e `.md` gerados aqui podem ser usados para auditoria, triagem e handoff multiagente.
