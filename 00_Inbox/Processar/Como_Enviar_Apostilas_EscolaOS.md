# Como Enviar Apostilas para o FabioOS

## Envio Ideal

Para cada apostila, informe:

```text
Disciplina:
Ano/Turma:
Bimestre:
Autoria: autoral / adaptada / terceiros / misto
Objetivo: aula / revisao / prova / banco de questoes
Pode virar markdown completo? sim / nao / duvida
```

## Se Nao Souber

Envie mesmo assim. O FabioOS classifica como `duvida` e coloca em triagem antes de gerar notas.

## Saida Esperada

- nota-fonte;
- mapa de conceitos;
- notas de conhecimento;
- roteiro de aula;
- revisao;
- questoes;
- gabarito quando aplicavel.
