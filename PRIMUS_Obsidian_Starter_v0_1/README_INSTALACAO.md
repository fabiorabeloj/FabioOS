# PRIMUS no Obsidian — Starter Pack v0.1

## Como usar

1. Extraia esta pasta dentro do seu vault `FabioOS`.
2. Mantenha a pasta principal em `30_Projetos/PRIMUS`.
3. No Obsidian, abra primeiro: `30_Projetos/PRIMUS/PRIMUS.md`.
4. Use os templates em `30_Projetos/PRIMUS/04_Templates`.
5. Se usar Dataview, os dashboards já listarão entradas, missões e ΔP automaticamente.

## Princípio

O Obsidian não é o motor final do PRIMUS. Ele é a interface editorial e cognitiva.

- `E` = notas catalogadas, tipadas e rastreáveis.
- `I` = missões, dungeons, encontros e pacotes de execução.
- `P` = WorldState, ΔP, histórico e consequências.

Nada entra como texto solto.
