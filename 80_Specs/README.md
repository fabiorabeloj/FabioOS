# 80_Specs

Camada de especificacoes executaveis.

Antes de implementar uma fase, agente, automacao ou integracao relevante, deve existir uma SPEC com objetivo, escopo, riscos, criterios de aceite e testes.

Compatibilidade:

Specs tecnicas ja existentes em `60_Sistemas/FabioOS/specs/` permanecem validas ate migracao planejada.
