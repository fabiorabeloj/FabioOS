# Obsidian

**Ferramenta de [[Segundo Cérebro]]**

---

## O que é?

Obsidian é um aplicativo de notas que armazena tudo em Markdown e permite [[Links Internos]] entre notas.

**Diferencial:** Tudo local. Você é dono dos dados.

---

## Por que Obsidian?

✅ Markdown — Formato aberto  
✅ Local — Seu computador, sua privacidade  
✅ Offline — Funciona sem internet  
✅ Rápido — Busca instantânea  
✅ Extensível — Plugins e temas  
✅ Barato — Grátis (ou $10-50/ano premium)  

---

## Recursos principais

### 📝 Notas
- Markdown básico
- Frontmatter (metadados)
- Blocos de código

### 🔗 Links
- [[Links internos]] — Conecta notas
- [[Backlinks]] — Quem referencia quem
- Graph view — Visualiza rede

### 🏷️ Tags
- #categoria — Organização
- #projeto — Filtro por projeto
- #status — Rastreamento

### ⚙️ Plugins
- Dataview — Queries em Markdown
- Tasks — Gerenciar tarefas
- Templater — Templates dinâmicos
- Obsidian Git — Sincronizar com GitHub

---

## Setup para FabioOS

1. **Abrir pasta** como vault
2. **Instalar plugins:**
   - Obsidian Git
   - Dataview
   - Tasks
   - Templater
3. **Criar estrutura de pastas**
4. **Sincronizar com GitHub**
