# Claude

**Modelo de linguagem como agente inteligente**

---

## O que é?

Claude é um LLM (modelo de linguagem grande) criado pela [[Anthropic]].

**Em FabioOS**, Claude funciona como:
- 🧠 Processador de pensamento
- ✍️ Gerador de conteúdo
- 🤔 Assistente de decisões
- 🤖 Agente autônomo

---

## Capacidades

### 💭 Análise Profunda
- Entender contexto complexo
- Raciocínio multi-passo
- Reconhecer padrões

### ✍️ Escrita
- Provas e revisões de Filosofia/Geografia
- Documentação técnica
- Criação narrativa (PRIMUS)
- Respostas estruturadas

### 🔍 Processamento de Contexto
- [[RAG]] — Com documentos
- [[Obsidian]] — Com seus conhecimentos
- APIs — Com dados externos

### 🎯 Execução de Tarefas
- [[Claude Code]] — Escrever scripts
- Transformações de dados
- Automações complexas

---

## Integração em FabioOS

### Interface
- [[OpenClaw]] — Chat direto
- [[Claude Code]] — CLI
- API — Programática via [[n8n]]

### Usos Específicos

**Escola:**
- Gerar estrutura de provas
- Revisar materiais
- Criar gabaritos comentados

**Trading:**
- Analisar gráficos
- Sugerir próximas ações
- Revisar trade journal

**IA & Programação:**
- Revisar código
- Sugerir otimizações
- Escrever scripts

**PRIMUS:**
- Criar eventos narrativos
- Descrever cenas
- Gerar NPCs
