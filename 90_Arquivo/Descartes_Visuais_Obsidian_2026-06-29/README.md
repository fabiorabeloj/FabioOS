# Descartes Visuais Obsidian - 2026-06-29

## Funcao

Preservar placeholders quase vazios criados pelo Obsidian antes da limpeza visual da raiz.

Nada foi apagado.

## Origem

Arquivos com nome `Sem titulo` foram encontrados como bases/canvas sem conteudo util.

## Regra

Nao criar conteudo novo aqui.

Se algum arquivo for recuperado, renomear com titulo real e mover para uma pasta canonica.
